## *** LIBRARY PROGRAM ***

### Requirements:

- Installed python3

### Description:

This small python program is used to save and display information of books in a text -file
acting as the database of a library.


### How to use the program:

The program can be used with python by starting the main.py and giving it a path 
to a .txt -file you want to use as the database of the library as a commandline argument.

FOR EXAMPLE: python main.py my_library.txt

When the program starts, you will be prompted to choose between:

- Adding a book to the database
- Displaying the current contents of the database
- Quitting the program
